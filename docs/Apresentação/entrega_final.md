# Entrega Final

## Tópicos

Esta apresentação corresponde à entrega final do projeto, reunindo tudo o que foi produzido pelo grupo ao longo da disciplina de Requisitos de Software. 

Seu objetivo é: 
- Apresentar de forma organizada todos os artefatos desenvolvidos
- Destacar a evolução do trabalho
- Evidenciar como cada etapa contribuiu para a construção da documentação completa do projeto.
