# Comprovação Informal

## Introdução

Este documento apresenta o contato com a equipe de desenvolvimento do SAE para validar os [protótipos](https://requisitos-de-software.github.io/2025.2-Grupo03/Validacao/prototipos/) feitos pela nossa equipe, referentes aos requisitos não implementados, e também para esclarecer dúvidas sobre a plataforma.

## Metodologia

O aluno [Vilmar José](https://github.com/VilmarFagundes) escreveu o texto do corpo do e-mail, buscando uma comunicação clara e respeitosa. Em seguida, ele enviou a mensagem para o professor **Vandor Rissoli**, que também é um dos desenvolvedores do SAE, no dia 17/11/2025. O e-mail foi respondido no mesmo dia, com o professor se disponibilizando a realizar a validação. A validação foi conduzida pelo próprio [Vilmar José](https://github.com/VilmarFagundes) e gravada por [Tiago Lemes](https://github.com/TiagoTeixeira-2005), apresentando o protótipo ao professor, que fez algumas observações
